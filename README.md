## Run Locally

Clone the project

```bash
  git clone https://github.com/kushagra-aa/stop-w.git
```

Go to the project directory

```bash
  cd stop-w
```

Install dependencies

```bash
  npm i
```

Development Build

Runs the app in the development mode.

```bash
  npm run tauri dev
```

The App Window will automatically appear
